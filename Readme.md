## 🌟 Top `4` Repositories

### 1. `DistribuFlow-Real-Time` - Cloud-Native Cyber Attack Detection System 🛡️
- **Repository:** [DistribuFlow-Real-Time](https://github.com/YassineOUAHMANE/DistribuFlow-Real-Time)
> Description: A **cloud-native streaming system** 🌐 that ingests live network data into a **Kafka cluster**, aggregates it with **Spark**, and uses a **machine-learning model** to classify cyber attacks in real time 🚨. Deployed on **GCP** and orchestrated by **Kubernetes**, with full observability through **Prometheus** and **Grafana** 📊, it turns raw traffic into actionable security alerts.

> - **Technologies/Tools:** Apache Kafka, Apache Spark, Machine Learning, GCP, Kubernetes, Prometheus, Grafana, Python.

### 2. `RecruitAI` - Intelligent Recruitment Assistant 🤖
- **Repository:** [RecruitAI](https://github.com/YassineOUAHMANE/RecruitAI)
> Description: An **intelligent recruitment assistant** 🧠 that analyzes your company's CV database and instantly identifies the best candidate profiles based on your job requirements. With a simple prompt, the **chatbot** searches, compares, and ranks applicants 🏆, helping HR teams save time and make smarter hiring decisions.

> - **Technologies/Tools:** LLM, RAG, NLP, Python, Vector Search, Chatbot.

### 3. `Distributed-Make` - Distributed Systems Testing ⚙️
- **Repository:** [Distributed-Make](https://github.com/YassineOUAHMANE/Distributed-Make)
> Description: A distributed systems project featuring tests for **PingPong** on **Grid5000** and **WordCount** workloads 🔬, measuring network performance and handling edge cases to evaluate the behavior and reliability of distributed computing environments.

> - **Technologies/Tools:** Grid5000, Distributed Computing, Network Performance Testing, Java/Python.

### 4. `Stock Sentiment Analysis Dashboard` - Financial Intelligence App 📈
- **Repository:** [Stock-Sentiment-Analysis-Dashboard](https://github.com/YassineOUAHMANE/Stock-Sentiment-Analysis-Dashboard)
> Description: A **Streamlit-based web application** 📊 that fetches the latest financial news and stock market data to help users make informed investment decisions. Combines **sentiment analysis** of news articles with real-time stock metrics for a comprehensive market overview.

> - **Technologies/Tools:** Streamlit, Sentiment Analysis, NLP, Stock Market APIs, Python, Data Visualization.
